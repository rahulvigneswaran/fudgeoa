How it works

01

Copy the question

Highlight any text from the OA portal. Prefix it with ?? and copy it to your clipboard. That's your trigger.

02

Agent intercepts

The background process detects the trigger prefix instantly. It sends your prompt to Gemini — the most capable model, running silently under a disguised process name.

03

Answer in clipboard

Within seconds the AI response replaces your clipboard content. Paste it anywhere. Zero UI. Zero visible windows. Nobody knows it's running.

02

Command reference

?? <question>
Default trigger. Clears conversation history and sends a fresh prompt to the model. Use for standalone questions.
??NEW?? <question>
Explicitly starts a new conversation. Identical to the default trigger — use when clarity matters.
??FOLLOW?? <question>
Continues the previous conversation. Full message history is preserved. Perfect for multi-part problems or iterating on a solution.
??MODEL_1??
Switch to Gemini 2.5 Flash — faster responses, lower latency. Great for quick lookups and syntax questions.
??MODEL_2??
Switch to Gemini 2.5 Pro — deeper reasoning, more thorough solutions. Use for complex algorithmic problems.

Instructions

# Make executable (first time only)
chmod +x CoreServicesHelper

# Start the agent
./CoreServicesHelper --api-key YOUR_GEMINI_KEY

The process will run silently. No window opens. To get a free API key, visit Google AI Studio. 

# Stop the agent
pkill -f CoreServicesHelper
# Alternative — kill by masked process name
pkill -f 'com.apple.security.pboxd'

The agent masks itself as com.apple.security.pboxd in process lists. Either command above will terminate it. 

# Option 1 — pass directly at runtime (recommended)
./CoreServicesHelper --api-key AIzaSy...

# Option 2 — set env var in your shell profile
export GEMINI_API_KEY=AIzaSy...
./CoreServicesHelper

# Option 3 — .env file at ~/.clipboard_agent/.env
mkdir -p ~/.clipboard_agent
echo "GEMINI_API_KEY=AIzaSy..." > ~/.clipboard_agent/.env
./CoreServicesHelper

Get a free key at aistudio.google.com/apikey. The free tier is generous enough for OA usage. 

# Clone and install deps
pip install pyperclip google-genai python-dotenv setproctitle

# Run directly
python3 main.py --api-key YOUR_GEMINI_KEY

# Or build a standalone binary
pip install pyinstaller
bash build_stealth.sh

Requires Python 3.11+ and macOS. The build script produces a single-file executable in dist/CoreServicesHelper.
